このドキュメントは、Webアプリ開発に詳しくない人に向けて、現代的なWebアプリケーション開発の概要を解説するものです。具体例として「tsundo-cleaner」というWebアプリケーションを使い、フロントエンドからバックエンドまで、様々な技術要素について詳しく説明しています。

## 技術スタック概要

tsundo-cleanerで使用されている主な技術は以下の通りです：

### フロントエンド
- **React 19**: UIコンポーネントの構築
- **TypeScript**: 型安全なコード開発
- **Vite**: 高速な開発環境とビルドツール
- **React Router**: SPAのルーティング
- **Axios**: HTTPクライアント
- **Recharts**: データ可視化
- **styled-components**: コンポーネントスタイリング

### バックエンド
- **Node.js**: サーバーサイドランタイム
- **Express**: Webアプリケーションフレームワーク
- **TypeScript**: 型安全なサーバーサイド開発
- **SQLite**: 軽量データベース
- **natural**: 自然言語処理（TF-IDF実装に使用）

これらの技術の選定理由と利点については、各セクションで詳しく解説しています。
